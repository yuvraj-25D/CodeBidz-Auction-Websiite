## 🌐 Overview  

**CodeBidz** is a dynamic auction-based web application designed to simulate a **real-time bidding ecosystem**.  

It provides users with a seamless experience to:  
- 💰 Participate in auctions  
- ⚡ Place competitive bids  
- 🔔 Receive live notifications  
- 📊 Manage credits and activity  

> 🎯 Built with a focus on **user engagement, real-time interaction, and intuitive UI/UX**

---

## ✨ Key Features  

### 🏷️ Auction System  
- Live auction listings with interactive bidding  
- Real-time bid updates and competition tracking  
- Transparent winner selection mechanism  

### 👤 User Authentication  
- Secure login/signup system  
- Role-based access (user/admin ready structure)  
- Personalized dashboard experience  

### 💳 Credit System  
- Virtual wallet for bidding credits  
- Tracks spending and user balance  
- Simulates real-world auction economy  

### 🔔 Notifications Engine  
- Real-time alerts for bids and auction updates  
- Keeps users engaged throughout the bidding process  

### 📊 Dashboard  
- Clean and modern UI with glassmorphism design  
- Displays user activity, credits, and role  
- Smooth navigation and user interaction  

---

## 🧠 System Architecture  

```bash
Frontend:
├── HTML (Structure)
├── CSS (Modern UI / Glassmorphism)
└── JavaScript (Logic & Interactivity)

Modules:
├── auth.js          # Authentication system
├── dashboard.js     # User dashboard logic
├── auctions.js      # Auction & bidding logic
├── notifications.js # Real-time alerts
├── utils.js         # Helper functions
└── config.js        # Configuration setup
```

## 🚀 How It Works
1. 👤 User logs in / signs up
2. 💳 Credits are assigned to the user
3. 🏷️ User explores available auctions
4. ⚡ Places bids in real-time
5. 🔔 Receives notifications on updates
6. 🏆 Highest bidder wins the auction
